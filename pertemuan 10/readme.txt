List	:
  Kelebihan
	- Dapat menyimpan data secara terurut
	- Dapat mengakses elemen melalui index secara langsung
	- Memungkinkan adanya data duplikat
  Kekurangan
	- Penggunaan memori yang besar, terutama ketika ada banyak data duplikat yang tidak penting

Set : 
  Kelebihan
	- Tidak ada data duplikat
	- HashSet pada java membutuhkan waktu konstan untuk operasi dasar
  Kekurangan
	- Tidak menyimpan data secara terurut
	- Tidak dapat mengakses elemen melalui index secara langsung

Map : 
  Kelebihan
	- Memungkinkan pengambilan nilai berdasarkan key
	- HashMap pada java membutuhkan waktu konstan untuk operasi dasar
	- Tidak akan ada duplikasi key
  Kekurangan
	- Tidak menyimpan data secara terurut
	- Memakan lebih banyak memori, karena harus menyimpan key dan value
